# libcsi  
simple library for connecting database and create query builder  
version 1.0  
* only support mysql/mariadb syntax

##installing
* create composer.json containing this
`{
	"minimum-stability": "dev",
	"require": {
		"php": ">=5.3.0",
		"libcsi/libcsi": "dev-master"
	}
}`
* run `composer install`  

thats it
